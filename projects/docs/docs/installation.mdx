Install the library from the **NPM repository**:

<Tabs>
<TabItem value="npm" label="npm" default>

```shell
npm install @hug/ngx-schematics-utilities --save-dev
```

</TabItem>
<TabItem value="yarn" label="yarn">

```shell
yarn add @hug/ngx-schematics-utilities --dev
```

</TabItem>
</Tabs>
